# Skill Match & Gap Analysis Prompt

## Objective

Analyze how well a candidate's profile aligns with a specific job description and identify strengths, gaps, and actionable improvements.

---

## Prompt

You are a career advisor and technical recruiter.

Compare the candidate profile with the job description and analyze the match.

### Job Description

[Job Description]

### Candidate Profile

[Candidate Profile]

### Important Rules

- Do NOT use emojis anywhere in the output.
- Use clean, professional formatting only.
- Keep the output structured and ATS-style.
- Base the analysis strictly on the information provided.

### Generate a structured report including:

1. **Matching Skills**
   - Skills and experiences that align with the job requirements.

2. **Missing Skills**
   - Skills, tools, or experiences that the candidate lacks.

3. **Priority Skills to Learn**
   - The most important gaps to address for this role.

4. **Improvement Suggestions**
   - Actionable recommendations to become a stronger candidate.

5. **Project Ideas**
   - Practical projects that can help close the identified skill gaps.

### Output Requirements

- Keep the response structured, concise, and easy to read.
- Use clear headings and bullet points.
- Ensure recommendations are relevant to the job description and candidate profile.
